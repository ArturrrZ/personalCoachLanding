# Personal Coach Landing Page

A modern, responsive landing page designed to showcase professional coaching services and convert visitors into clients.

## Project Overview
This landing page serves as the digital storefront for a personal coaching business, featuring service offerings, coach credentials, and clear calls-to-action for booking sessions.

## Key Features
- Responsive design for all devices
- Service descriptions and pricing
- Client testimonials section
- Contact form and booking integration
- SEO-optimized structure

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Django
